# Paper Entry Template

Use this template when adding a new paper.

## Markdown Table Row

```markdown
| Method | Paper | Venue/Year | Task | Problem | Code |
|---|---|---|---|---|---|
| <method> | <paper title> | <venue, year> | <AS/AC/SM/AG> | <SOP/MOOP/...> | <url or -> |
```

## CSV Row

```csv
id,title,venue,year,paradigm,task,problem_type,link,code
<id>,<title>,<venue>,<year>,<RL/SL/NE/ICL>,<AS/AC/SM/AG>,<SOP/MOOP/...>,<paper_link>,<code_link_or_dash>
```

## ID Convention

`<year>-<first_author>-<short_method_or_keyword>`

Example: `2024-guo-rl-das`
